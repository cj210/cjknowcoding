What is HTML?

HTML stands for Hyper Text Markup Language and is the basic structure and building blocks of any web site

HTML tags

HTML tags are element names surrounded by angular brackets. In HTML we use:

  1. Start tag: Also called opening tag   ex: <p>
  2. End tag: Also called closing tag   ex: </p>

Thus a basic HTML format can be described as follows:

<!DOCTYPE html>
<html>
 

<head>
  

<title>Title of the page </title>
  </head>
  <body>
    <h1>Main heading</h1>
     <p>Content of the page</p>
     </body>
    </html>
