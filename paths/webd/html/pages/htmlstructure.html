The basic stucture of HTML

HTML describes the structure of a web page. And the stucture of HTML consists of series of elements. These elements will convey to the browser, how the content of the page is expected to be displayed. A simple HTML document can be represented by: 
                        <!DOCTYPE html>
                        <html lang="en">
                        <head>
                            <meta charset="UTF-8">
                            <meta name="viewport" content="width=device-width, initial-scale=1.0">
                            <title>Document</title>
                        </head>
                        <body>
                            <h1>This is the main heading</h1>
                            <p>This is the text content</p>
                        </body>
                        </html>
Lets depict the above code line by line
Line Explaination
<!DOCTYPE html> This is the declaration that defines this document as a HTML5 documetn
<html lang="en"> html is the root element of the page with an attribute lang, which decribes the language of the content as "en", which is english.
<head> All the meta information of the web page is confined in the head tag.
<meta charset="UTF-8"> Meta is a HTML tag that contains metadata about a web page, or more specifically, descriptors that tell search engines what type of content a web page contains that is hidden from display. charset is an HTML attribute that defines the character encoding for your browser to use when displaying the website content. utf-8 is a specific character encoding. Basically, this line tells the browser to use the utf-8 character encoding when translating machine code into human-readable text and vice versa to be displayed in the browser.
<meta name="viewport" content="width=device-width, initial-scale=1.0"> The viewport is the user's visible area of a web page. The viewport varies with the device, and will be smaller on a mobile phone than on a computer screen. This line gives the browser instructions on how to control the page's dimensions and scaling. The width=device-width part sets the width of the page to follow the screen-width of the device (which will vary depending on the device). The initial-scale=1.0 part sets the initial zoom level when the page is first loaded by the browser.
<title> Title element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
<body> Body element lement defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<h1> h1 element describes the largest and most important heading on the page.
<p1> p element contains the text. It defines the paragraph content.
