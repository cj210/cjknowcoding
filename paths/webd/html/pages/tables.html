Tables

Important tags of a table
Tag Description
table Defines a table
caption Defines caption for a table that is diplayed on the top of the table
tr Defines a table row. Immediate child of table
th Defines a header cell inside a table. Immediate child of tr
td Defines a cell inside a table. Immediate child of tr
colgroup Specifies a group of one or more columns in a table
col Specifies properties of each column in a colgroup
thead Groups the header content in a table
tbody Groups the body content in a table
tfoot Groups the footer content in a table
