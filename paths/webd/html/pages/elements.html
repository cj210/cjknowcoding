Tags, Elements and attributes.

Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags. Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag. And attribute is used to define the character of an HTML element. It always placed in the opening tag of an element. It generally provides additional styling (attribute) to the element.

Elements and their attributes

Below table shows the basic elements, their functions and possible attributes.
Element Function Attributes
h1 - h6 Headings for page, with h1 being the most important, coming down to h6 class, id , title
p paragraph element to contain text in the web page. class, id, title
a anchor tag, acts as hyperlink to another page which can be absolute or relative href - This attribute take a complete url or a file from a location. Id of an element can also be provided href="html_demo.html#C4". To email href="mailto:someone@example.com". target is how the link open the page. target can be _self, _blank, _top or _parent
img renders any image src - can be absolute or relative. alt - is an alternative text when image doesn't load. width and height
b, strong, i, em, mark, small, del, ins, sub, sup formatting elements cite (for del and ins)
blockquote Shows text as block (next to heading) cite - specifies url that explains the quote
q, cite, address, abbr, bdo abbr is abbreviation, address and cite are just like i and em, bdo get the text in reverse order and q for quotation marks title for abbr, dir for bdo (rtl)
link Links to a extenal style sheet rel="stylesheet" href="url/path"
pre retains the format inside the tags
hr horizontal row
br new line
