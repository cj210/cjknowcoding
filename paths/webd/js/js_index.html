What is Java Script?

Java Script is a High-level, object-oriented, multi-paradigm, multi paradigm programming language. Now lets see what the terms in that sentence mean.

  • High-level: We don't have to worry about complex stuff like memory management.
  • Object-oriented:Based on objects for store most kind of data.
  • Multi-Paradigm: We can use different styles of structuring for our code.
  • Programming: Instruct computer to do things.

What is the role of JS in web develpment?

The modern web pages are a result of HTML, CSS and Java Script. Their roles can be explained as follows:

  • HTML - The Noun: Html is the basic building blocks of a web page which has all the content like the paragraphs, images etc.
  • CSS - The Adjective: CSS is how the content is presented. Its the basic style of the web page
  • JS - The Verb: This adds the dynamic and interactive effects to the web page and manupulates the content of CSS. The real programming language of the web.

How is it used?

JavaScript can be declared with the <script></script> tag in an HTML file. All the code goes into the tag. Alternatively, we can use a seperate file, .js file, and add it to the HTML file with src attribute of an empty <script></script> tag. In general this tag is added just above the closing body tag </body>
