Linux command structure:

cmd -options arguments

Types of commands:

  1. Executable programs: These are compiled binary files present in directories like /bin, /usr/bin, /usr/local/bin etc.
  2. Built in shell commands: These commands are part of shell.
  3. Shell functions
  4. Aliases
Useful commands
Command Usage
man Displays manual for a particular command. ex: man ls. Can also be used to display manual for a key word using man -k key_word
which To find the exact location of an executable. This wont work for built in commands or aliases.
help Some commands dont have man pages as they are built directly into shell. Documentation for such commands can be displayed by running cmd --help or cmd -h
id Displays user and all the groups that the user is part of.
groups Displays all the groups that the user is part of.

Permissions in Linux

The permission to read (r), write (w) and execute (x) a particular file / directory can be alter for users (u), groups (g), others (o) and together as all (a = u + g + o). This can be done by setting (=), adding (+) and removing (-) rwx for ugoa using chmod command.

examples:
  • chmod u=rwx file.txt
  • chmod g-rx file.txt
  • chmod o+x file.txt
  • chmod a-w file.txt
Useful commands
Command Usage
chgroup Changes the group
chown chown command is used to change the file Owner or group. chown [OPTION]… [OWNER][:GROUP] FILE…
addgroup adds a new group. addgroup groupname
adduser Adds user to a particular group. adduser username groupname.
File creation mask:
  • It determines the default permissions. 777 for directories and 666 for files.
  • umask 022 This command set permission as 755 which is 777-022
  • unmask command displays 4 digits instead of 3 (ugo). The first digit is for special mode which can be setuid / setgid / sticky.
  • umask is opposite of chmod
