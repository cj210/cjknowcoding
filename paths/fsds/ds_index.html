Welcome everyone to your the guide to the basics of SQL...

General Database concepts:

  1. Instance: An instance is an a installation of a SQL Server. Every time a SQL server is created and SQL is installed on it, its a new instance of SQL. Each instance has a unique name on the network.
  2. Database: Organised collection of data. Its a container to store and organise data.
  3. Table: A collection of data which is organised into rows and columns.
  4. Schema: Is a physical grouping of tables in a logical way.
  5. View: Views are like virtual tables that combine different selections of rows and columns.

Querying and SELECT statement:

Literal Select Statements


        SELECT 'Kalicharan'
        SELECT 'Kali', 'Charan', 'CJ'
        SELECT (2+5)-(12/4)+(5*12)

Basic Select statement:


        SELECT col1, col2, col3 ...., coln FROM database.schema.table
    
        /*Specifying database name is not necessary, if it is same as the connected database. Following examples are from a table Person in the schema Person
        SELECT Firstname, Lastname FROM Person.Person
        SELECT TOP 100 Firstname, Lastname FROM Person.Person
        SELECT TOP 10 PERCENT Firstname, Lastname FROM Person.Person
        SELECT TOP 10 PERCENT Firstname AS [Given Name], Lastname AS "Sur Name" FROM Person.Person
        SELECT TOP 10 * FROM Person.Person

The WHERE clause:

WHERE clause is used to filter the rows of data based on some criteria. 


        SELECT col1, col2, col3 ...., coln FROM database.schema.table WHERE somecolumn someComparisionOperator someValue

        SELECT * FROM Production.product WHERE ListPrice > 10
