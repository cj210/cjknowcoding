General Database concepts:

  1. Instance: An instance is an a installation of a SQL Server. Every time a SQL server is created and SQL is installed on it, its a new instance of SQL. Each instance has a unique name on the network.
  2. Database: Organised collection of data. Its a container to store and organise data.
  3. Table: A collection of data which is organised into rows and columns.
  4. Schema: Is a physical grouping of tables in a logical way.
  5. View: Views are like virtual tables that combine different selections of rows and columns.
